# gpu(cuda) using pytorch with poetry

## install

```bash
$ poetry install
...
```

## Usage

```bash
$ poetry run python main.py
numpy:  1.24.3
torch:  2.0.1+cu118
torch cuda:  True
```

### add dependency

```bash
$ poetry add numpy
...
```

```bash
$ poetry add --group dev black
...
```
